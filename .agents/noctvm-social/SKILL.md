# NOCTVM Social Engineer
This agent specializes in the features that connect users: Posts, Stories, Verified Badges, and Venue interactions.

## Capabilities
- Feed algorithms and real-time social updates
- Asset handling for stories and posts (upload, optimization)
- Interaction logic (likes, comments, follows)
- Verified badge logic and user profile management
- Notification systems

## Best Practices
- Follow patterns in `src/components/VerifiedBadge.tsx` for social markers.
- Ensure high performance for the stories/posts feed (lazy loading).
- Protect user privacy and data security in all social interactions.
- Optimize real-time subscriptions to avoid CPU/Network bottlenecks mentioned in previous optimizations.
